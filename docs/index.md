# com-pac: Center Of Mass - Principal Axes Calculator

A Python package for transforming the Cartesian coordinates of a molecular structure
into the center-of-mass, principal-axes coordinate system.

Originally written [Andrew N. Owen](https://orcid.org/0000-0001-5903-1651) during his
PhD research. Updated for use in continuing collaborations.

## Getting Started

Install the package with 

```
python3 -m pip install com-pac
```

Then run

```
com-pac --help
```

to get started.
